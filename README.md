# Coach 360 Mujer

**Más claridad. Más poder. Más tú.**

App de coaching personal para mujeres con IA.

## Setup

1. Copia `.env.local.example` a `.env.local` y completa las variables
2. `npm install`
3. `npm run dev`

## Stack

- Next.js 14
- Supabase (auth + database)
- Anthropic API (Clara - coach IA)
- Vercel (deploy)
